This archive contains the code examples for the book:

Python 3 Web Development Beginner's Guide
Michel Anders
Packt Publishing
978-1-84951-374-6

Chapter 2 (Spreadsheet)
Chapter 3 (Tasklist, filesystem based)
Chapter 4 (Tasklist, SQLite based) 
Chapter 5 (Books, simple)
Chapter 6 (Wiki)
Chapter 7 (Books, more elaborate)
Chapter 8 (CRM, first steps)
Chapter 9 (CRM, extensions including those descibed in Chapter 10)

Each directory contains it own README.txt file that lists the files and their purpose. All examples expect Python 3.x and CherryPy 3.2.x to be installed correctly as described in the book. Note that the code samples featured in chapter 10 may be found in the "Chapter 9" directory.
